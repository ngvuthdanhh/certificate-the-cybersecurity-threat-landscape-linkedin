# Project: Threat Intelligence Report

## Objective
Develop a structured cybersecurity threat report based on current events.

## Tasks
1. Select a recent cyberattack from a trusted source (CISA, CrowdStrike, FireEye, etc.).  
2. Analyze the attack method, threat actors, and impact.  
3. Map the attack to the **MITRE ATT&CK framework**.  
4. Recommend mitigation and detection strategies.
