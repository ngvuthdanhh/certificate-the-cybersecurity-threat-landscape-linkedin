# Project: Phishing Awareness Campaign

## Objective
Design a cybersecurity awareness campaign focused on phishing prevention.

## Tasks
1. Draft phishing simulation emails for training purposes.  
2. Create an employee awareness checklist.  
3. Develop a short presentation (slides) on how to detect phishing attempts.  
4. Measure effectiveness through a mock phishing test (optional).  

## Deliverables
- Awareness campaign materials (slides, checklist).  
- Simulation results (if performed).  

## Key Learning
Security is not only about technology but also **human awareness and behavior**.
